# Закрывающий тег

Учебный проект — одностраничный сайт-история о пути обучения вёрстке.
Проект выполнен с акцентом на семантику HTML, архитектуру CSS, анимации и аккуратное взаимодействие с JavaScript.

## Описание

Сайт представляет собой последовательность карточек, каждая из которых символизирует этап обучения.
Главный визуальный образ — курсор-персонаж, проходящий путь от неопределённости к уверенности.

Проект демонстрирует:

- современную адаптивную вёрстку;
- работу с CSS-переменными;
- фильтры изображений;
- SVG-анимации;
- интерактивные элементы (лайки, модальное окно).

## Используемые технологии

- **HTML5**
  - семантическая разметка
  - `<picture>`, `<dialog>`, SVG `<use>`
- **CSS**
  - CSS Variables
  - Flexbox
  - `clamp()`, `aspect-ratio`
  - `@supports`
  - CSS-анимации и transitions
  - `filter` для изображений
- **JavaScript**
  - обработка кликов
  - управление состоянием лайка
  - работа с классами
- **SVG**
  - пиксельное сердце с анимацией
- **Шрифты**
  - Inter (variable)
  - Press Start 2P
